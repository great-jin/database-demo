## Database-demo
如何能够快速了解一个数据库？除了查看基础的文档没有什么比直接看 ``Demo`` 更容易上手了。

### Talk is cheap, show me your code.

本工程包含当下流行的数据库与中间件使用样例。

项目中针对每一种数据库分别提供 ``Maven`` 与 ``Spring Boot`` 项目集成示例，助你快速上手。
